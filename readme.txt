Project Name: About Us Page

Description:
This is a simple About Us page developed using HTML and CSS. It follows a mobile-first approach and does not use any CSS frameworks like Bootstrap.

Files Included:
- index.html: The main HTML file.
- styles.css: External CSS file for styling.
- ai_usage.txt: Declaration of AI usage.
- readme.txt: Instructions for running the project.

How to Run:
1. Open index.html in any modern web browser.
2. Ensure the styles.css file is in the same directory.
3. No additional setup is required.

Hosting Instructions:
- You can deploy the project using GitHub Pages, Netlify, or Vercel for hosting.
